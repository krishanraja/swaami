## Core Philosophy

Swaami isn't about competition. It's about **belonging**. Our gamification creates a **positive-sum feedback loop** where helping others makes you feel good, earns you social capital, and gives you the ability to ask for help yourself.

**The Golden Rule**: Every interaction should reinforce:
> "I belong here. I matter. My neighborhood is better because I'm in it."

## 1. The Credit Economy

### How Credits Work
- **Earn**: Complete a task â†’ +1 credit (minimum), more for longer/harder tasks
- **Spend**: Post a need â†’ costs 0-1 credit depending on complexity
- **Starter**: Every new neighbor gets 5 credits free
- **Overflow**: Credits cap at 50 to encourage spending, not hoarding

### Why This Works
- **Reciprocity engine**: You can't just take â€” you must give to receive
- **No real money**: Removes transactional coldness, keeps it neighborly
- **Low stakes**: Nobody feels exploited or taken advantage of

## 6. Micro-Celebration Design

### Moments That Matter

1. **Task Completion**
   - Confetti burst (small, elegant)
   - Haptic feedback on mobile
   - "+1 credit" animation
   - "You helped [Name]!" confirmation

2. **Tier Up**
   - Full-screen celebration (2 seconds)
   - Badge reveal animation
   - Share prompt (optional)

3. **First Help**
   - Special "First Helper" badge
   - "Welcome to the community" message
   - Tutorial complete â†’ Feed unlocks

4. **Streak Milestones**
   - Badge animation
   - Subtle confetti
   - "Keep it going!" encouragement

### Animation Principles
- **Quick**: Never block the user
- **Subtle**: Delight, don't distract
- **Meaningful**: Only celebrate real achievements
- **Optional**: Users can reduce animations in settings

## 7. Notification Strategy

### Smart Nudges (Not Spam)

| Trigger | Message | Timing |
|---------|---------|--------|
| Streak at risk | "Your 7-day streak ends in 6 hours!" | 6 hours before expiry |
| Nearby need | "Sarah needs help 200m away" | Only if user is active |
| Task completed | "Thanks! You earned 1 credit" | Immediately |
| Tier approaching | "2 more helps to Trusted Neighbor!" | After each task |
| Idle 7 days | "Your neighbors miss you" | Once per week max |

### Notification Rules
- Max 3 per day
- Never at night (respect sleep)
- One-tap disable for each type
- "Quiet mode" for vacations

## 11. Metrics That Matter

### Success Indicators

| Metric | Target | Why |
|--------|--------|-----|
| 7-day retention | 60%+ | Users return after first week |
| Tasks per active user | 2+/month | Regular engagement |
| Circuit formation rate | 20%+ | Real relationships forming |
| Tier progression | 30% reach T2 | Users see value in long-term |
| NPS | 50+ | Would recommend to neighbors |

### Red Flags

| Metric | Warning | Action |
|--------|---------|--------|
| Unmatched needs | >40% | Improve matching, seed helpers |
| Streak breaks | >80% | Too aggressive, adjust timing |
| Credit hoarding | >30 avg | Encourage spending, lower need cost |
| Tier stagnation | <10% progress | Add smaller milestones |
