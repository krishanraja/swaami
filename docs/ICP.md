# Swaami Ideal Customer Profile (ICP)

## Primary Persona: The Urban Helper

### Demographics
- **Age**: 25-55
- **Location**: Urban/suburban Australia (Sydney, Melbourne focus)
- **Housing**: Apartment dwellers, townhouse residents
- **Tech Comfort**: Uses smartphone daily, familiar with apps

### Psychographics
- Values community connection
- Time-poor but willing to help
- Prefers local, face-to-face interactions over anonymous platforms
- Concerned about safety but not paranoid
- Seeks meaningful micro-contributions

### Behaviors
- Uses neighborhood WhatsApp/Facebook groups (frustrated by noise)
- Has offered help but struggled to find neighbors who need it
- Wants to contribute but doesn't have hours to volunteer
- Trusts verified systems over anonymous platforms

## Secondary Persona: The Busy Parent

### Profile
- Working parent with limited time
- Needs quick help (groceries while sick, tech setup)
- Willing to return favors when able
- Values trust and safety above all

### Pain Points
- Can't always wait for family/friends
- Doesn't want to burden neighbors with big asks
- Needs reliable, verified help

## Tertiary Persona: The New Resident

### Profile
- Recently moved to neighborhood
- Doesn't know neighbors yet
- Eager to build community connections
- May need help navigating local area

### Pain Points
- No existing network to ask for help
- Wants to meet neighbors naturally
- Unsure who to trust

## Market Size (Australia)

- **Urban apartment dwellers**: ~4M households
- **Target addressable**: 500K households in Sydney/Melbourne
- **Initial focus**: 5 neighborhoods (Surry Hills, Newtown, St Kilda, Brunswick, Fortitude Valley)

## Customer Journey

```
Awareness → Download → Onboarding → First Task → Trust Building → Regular Use
    │           │           │            │             │             │
    └───Ad/WOM──┴──App───────┴──Verify────┴──Post/Help──┴──Tier Up────┴──Advocate
```

## Success Indicators

| Stage | Metric | Target |
|-------|--------|--------|
| Acquisition | Sign-up completion | >80% |
| Activation | First task posted | Within 7 days |
| Retention | 2nd task | Within 30 days |
| Revenue | Credit purchase | Month 3+ |
| Referral | Invite sent | 1 per active user |

## Anti-Personas (Who We Don't Serve)

- **Gig economy workers** looking for paid tasks
- **Commercial services** seeking customers
- **Users seeking high-value tasks** (moving, renovations)
- **Anonymous users** unwilling to verify identity
